# 人工智能学院2026年普通招考博士研究生 （第一批次）综合考核工作通知
根据学校《关于做好2026年招收攻读博士学位研究生工作的通知》和《人工智能学院2026年普通招考博士研究生招生工作实施细则》，现将我院2026年普通招考博士研究生第一批次综合考核录取相关事项通知如下：
一、组织领导
我院研究生招生工作领导小组全面负责我院2026年普通招考博士研究生综合考核工作。
二、招生计划
（一）学术学位招生计划
学院2026年计划招收学术学位博士研究生23人，已招收直博生10人。
（二）专业学位招生计划（不含卓工专项计划）
学院2026年计划招收全日制专业学位博士研究生7人。
（三）卓工专项计划候选人及考核通知请关注卓工学院专项简章。
**学院2026年博士招生计划实际录取人数根据最终下达招生计划数及生源情况可能调整，以实际拟录取结果为准。学院根据第一批次招生计划完成情况，确定是否开展第二批次选拔。**
三、外语综合水平考核
我院根据外语水平免试条件进行审核，免试考生审核结果已在博士招生管理系统中公布，非免试考生须参加2025年12月21日武汉大学2026年博士外语水平考试，成绩达到60分。
四、候选人选拔
我院考核专家组根据申请者提交的申请材料，进行集体审核评议，对考生的学习能力与学科基础、学术研究（研发）能力与创新成果、培养潜力进行审核评议，做出综合评价结论。依据综合评价成绩，按不超过招生计划数1:3的比例确定候选人，经学院招生领导小组审核同意，确定候选人名单（见附件1）。
确定参加综合考核的候选人请于**2026年1月4日17：00前实名申请加入“WHUAI2026年博士综合考核群（第一批次）”（QQ群号：933628109）。入群验证申请格式：“姓名-报考专业”** ，后续通知会在群内发布，请及时关注群消息和保持通讯畅通。
五、综合考核
（一）综合考核方式
学院2026年博士研究生的综合考核采取线下方式进行。
（二）综合考核时间安排
2026年1月8日 09:00—11:00 报到、资格审查
2026年1月8日 14:30—16:30 笔试
2026年1月9日 08:30—18:00 综合面试
具体安排QQ群内通知，请保持通讯畅通。
（三）现场报到及资格审核
参加综合考核的候选人在报到时须提供以下材料：
1.往届生（已取得硕士学位人员）须交验本人最后学位、学历证书和身份证等原件；应届硕士生须交验硕士研究生证、学信网学籍认证报告和身份证等原件；
2.在境外教育科研机构获得硕士学位、学历的考生，须出具教育部留学服务中心的认证书原件或者学校出具的预计毕业时间的证明原件。
3.提交以下材料复印件：
（1）按顺序装订（左上角）：身份证（正反面复印在同一页面），最后学历证书、最后学位证书（应届硕士生提交学生证及学信网学籍认证报告；获得境外学位、学历的考生，提交教育部留学服务中心的认证书复印件或学校出具的预计毕业时间证明复印件）。
（2）单独提交：《2026年武汉大学博士研究生考生信息卡》。须本人仔细核对，并粘贴照片及签名。
（3）单独提交：《考生诚信综合考核承诺书》（附件2，本人亲笔签名）。
**4.考生须于报到时携带U盘，将个人汇报PPT拷贝至工作人员处，报到结束后不再接受拷贝或更换。**
学院将核对考生的报考信息，按报考条件严格审查考生报考资格，资格审核未通过的考生不得参加综合考核和录取。不符合报考条件或提供虚假信息者，取消其综合考核资格。**考生未按时到现场报到或参加综合考核者，视为自动放弃综合考核资格。**
综合考核收费180元/每人，收费办法及要求由培养单位按学校有关规定执行。
（四）综合考核内容
综合考核内容主要包括笔试和综合面试。
1.笔试。主要考查候选人是否具备本学科博士研究生应具备的专业基础知识。考试时间为2小时，满分为100分，笔试采用闭卷方式。
2.综合面试
（1）外语水平（不超过3分钟）。在面试中进行英语听力口语测试。主要考查候选人是否具备本学科博士研究生应具备的外语应用能力。候选人须准备2分钟以内英语自我介绍，并回答专家提问。
（2）个人汇报。候选人向考核专家组汇报本人基本情况，硕士期间科研经历、科研成果等，陈述对拟从事研究的学科领域及研究方向的认识、研究思路和展望等。汇报采用PPT形式，汇报时间不超过7分钟。
（3）考核专家组对候选人基础知识、综合运用所学知识分析问题、解决问题的能力及学术研究的敏锐度、道德品质等综合素质进行全面考察。
（4）思想政治素质考核。考察考生的政治态度、思想表现、学习（工作）态度、道德品质、遵纪守法、诚实守信等方面。特别注重考查考生的科学精神、学术道德、专业伦理等方面情况。
（五）综合考核总评成绩计算
综合考核总评成绩=笔试成绩×30%+面试成绩×70%
综合考核成绩按百分制计算，综合考核总评成绩不及格（低于60分）不予录取。
六、录取原则
坚持“择优录取、保证质量、宁缺毋滥”原则，学院招生工作领导小组根据招生计划、考生综合考核总评成绩确定拟录取名单并公示。**综合考核不及格（低于60分）的考生不予录取，思想政治素质和品德考核不合格者不予录取** 。
七、其它事宜
本实施细则报经学校研究生院审核通过后公布实施，未尽事宜按学校相关规定执行。
