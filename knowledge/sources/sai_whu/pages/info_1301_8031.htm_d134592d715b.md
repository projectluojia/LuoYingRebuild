# 【专业介绍】数学与应用数学（智能科学）
数学与应用数学（智能科学）专业是武汉大学面向国家新一代人工智能发展战略需求，依托数学与统计学院、人工智能学院的学科优势，于2025年设立的新型交叉专业，首次招生。本专业以智能科学为牵引，培养具有坚实数理基础、掌握智能科学前沿关键技术的复合型拔尖人才。作为强基计划重点专业，本专业采取本研衔接培养模式，由人工智能学院支撑专业培养，实现“数理基础+智能技术”的深度融合。
【专业名称】
本科阶段专业名称：数学与应用数学
本科阶段专业代码：070101
研究生阶段专业名称：智能科学与技术
研究生阶段专业代码：1405
【培养目标】
本科培养目标：本专业通过对数学基础与智能科学专业能力的培养，塑造具有科学探索精神和创新实践能力的拔尖创新人才。
研究生培养目标：秉持“极基础、强技术、重交叉、促创新”的培养理念，培养具备浓厚家国情怀、扎实数理基础和创新科学思维，能够在数学与智能科学领域取得突破性进展，推动融合创新的领军型人才。
【特色课程】
| 数理基础  | 工科数学分析、工科高等代数、工科概率论与数理统计、信息科学中的物理学、离散数学、常微分方程、数值分析、数据科学、计算方法与优化、随机过程、量子信息与量子计算基础  |
| --- | --- |
| 核心技术  | 高级语言程序设计、计算机系统导论、数据结构与算法、现代信号处理、机器学习、深度学习与强化学习、多模态与大模型、数据库系统、大数据技术与应用、自然语言处理、计算机视觉、智能机器人、智能计算系统、多智能体系统  |
| 实践实训  | 人工智能基础实训、智能计算系统实践、人工智能交叉研究实践、人工智能综合项目实践  |
【培养特色】
1.极基础、强技术的课程体系
课程设置覆盖数理基础（工科数学分析、工科高等代数、工科概率论与数理统计、离散数学、信息科学中的物理学等）、人工智能核心领域（多模态与大模型、机器学习与深度学习、自然语言处理等）以及交叉课程（计算方法与优化、数据科学、随机过程、量子信息与量子计算基础等）。
2.重实践、项目制的培养模式
在强化数理基础和人工智能核心技术的同时，通过人工智能基础实训、智能计算系统实践、人工智能交叉研究实践、人工智能综合项目实践等项目制集中实训课程驱动学生掌握人工智能关键技术与解决真实场景问题的能力。
3.促创新、多层次的培养平台
从本科阶段起，学生即可参与各级科研项目，通过“学科竞赛-科研训练-产业实训”的培养模式，全方位促进交叉科研训练。通过与人工智能领军企业、国家级科研院所建立战略合作关系，共建联合实验室和实习实践基地，为学生提供多层次、全方位的实践创新平台。
【毕业生就业与深造】
学校对符合培养要求的强基计划学生实行本研衔接培养。研究生毕业后，鼓励学生在国家重大战略领域从事人工智能研究、开发、应用以及管理等相关工作。
