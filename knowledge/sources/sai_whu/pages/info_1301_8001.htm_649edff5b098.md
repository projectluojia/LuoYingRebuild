# 【专业介绍】人工智能（自强班）
人工智能是引领未来的战略性技术，也是国际竞争的新焦点、经济发展的新引擎、社会建设的新机遇，在国家战略中占据着极为重要的地位。人工智能以数理为根基，融合计算机科学、信息科学、生命科学、社会科学、认知科学、神经科学、统计学、心理学等多学科知识，构建独特的跨学科培养体系。主要研究方向包括数据科学、机器学习、计算机视觉、自然语言处理、多模态大模型、具身智能、群体智能、科学智能与社会智能等前沿领域，致力于探索人工智能基础理论和关键技术，推动智能技术在生物医药、时空智能、智能制造、水利电力、工业设计、人文社科、金融科技等领域的创新应用。
学院构建了完整的人工智能特色培养体系，依托全国重点实验室、国家工程技术研究中心、前沿科技创新中心等高水平科研平台，与国内外顶尖高校、研究院所和领军企业建立广泛的合作关系；通过产教融合、科教协同的培养模式，培养学生扎实的理论基础和解决实际问题的创新能力。
人工智能自强班是2025年3月获批的校级试验班，首次招生，本科阶段采用弹性学制，鼓励学生提前修满学分。学有余力的学生在本科阶段可以提前修习研究生阶段课程，获得研究生阶段课程学分。
【培养目标】
培养具备浓厚家国情怀、扎实数理基础、创新科学思维、全栈实践能力、卓越协作精神，系统掌握人工智能基础理论、方法与技术，具有良好的思维和创新能力，能够从事数据科学、机器学习、计算机视觉、自然语言处理、多模态大模型、具身智能、群体智能、科学智能与社会智能等方向的研究、开发、应用与管理的行业领军人才。
【特色课程】
| 数理基础  | 工科数学分析、工科高等代数、工科概率论与数理统计、信息科学中的物理学、数据科学、随机过程、量子信息与量子计算基础  |
| --- | --- |
| 核心技术  | 高级语言程序设计、计算机系统导论、现代信号处理、数据结构与算法、机器学习、计算方法与优化、离散数学、数据库系统、大数据技术原理、深度学习与强化学习、多模态与大模型、计算机视觉、自然语言处理、智能机器人、智能计算系统、智能交互技术、多智能体系统、智能语音处理  |
| 实践实训  | 人工智能基础实训、智能计算系统实践、人工智能交叉研究实践、人工智能综合项目实践、科研训练  |
| AI+X  | 医学人工智能、AI for Math、时空智能学、低空智能感知与应用技术、社会计算与社会智能  |
【培养特色】
1.AI+X双导师指导
校长张平文院士领衔数理基础、智能技术、科学智能、社会智能导师团队，精心打造面向人工智能拔尖人才培养课程群，引进国际一流大学的前沿课程与先进教学模式。全栈式贯通AI+X交叉培养，采用AI+X双导师指导制，为每位学生配备由人工智能领域专家与X学科（空天信息、生命科学、人文社科等）知名学者组成的高水平导师组，实现多维度、立体化指导。
2.定制化拔尖创新人才培养
定制化培养方案：定制化AI数理基础课程筑牢发展根基；体系化AI核心技术课程掌握核心本领；模块化AI+X专业选修课程突破学科壁垒；项目制AI实践实训体系激发无限可能。定制化培养模式：鼓励学生提前毕业，本科阶段可选修研究生课程；灵活学分免修机制；享受雷军教育基金支持。
3.对接国家人工智能学院
学生可直通北京、上海、深圳国家人工智能学院，实现“前沿研究-国家需求-产业应用”全链条培养。学院积极与人工智能领军企业、国家级科研院所建立战略合作关系，共建联合实验室和实习实践基地，为学生提供多层次、全方位的实践创新平台，着力提升学生解决复杂工程问题的能力。推动学术研究与产业实践深度融合，实现学术链与产业链的双向赋能。
【毕业生就业与深造】
就业通道：
毕业生就业方向广泛，可在科技企业，如华为、小米、腾讯、百度、阿里、字节、比亚迪、宇树、深度求索（deepseek）等，从事数据科学、机器学习、计算机视觉、多模态大模型等人工智能专业方向的研究、开发、应用与管理；也可在人工智能赋能领域（如智能医疗、智慧城市、时空智能、情报智能等），从事人工智能研究、开发、应用以及管理等相关工作。
深造通道：
毕业生可遴选进入国家人工智能学院联合培养，也可赴国内外顶尖名校深造。
【招生方式】
高考招生+进校二选（全校）
