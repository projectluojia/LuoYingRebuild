# 学院“十五五”规划专家咨询会顺利召开
5月9日下午，学院在A211会议室组织召开学院“十五五”规划专家咨询会。会议特邀相关学科与行业领域资深专家为学院发展把脉问诊、建言献策。学院领导班子成员及规划编制工作组成员参会，学院党委书记高卫松主持会议。
会上，执行院长夏桂松代表学院作规划编制汇报。他系统梳理了学院当前的发展现状、办学特色及学科建设与人才培养成效，重点围绕“十五五”规划的编制背景、总体思路、核心目标、重点任务及文本框架进行了全面解读，向专家详实呈现了规划的战略考量与顶层设计。
在专家指导环节，与会专家在认真审阅规划文本、听取情况介绍的基础上，紧扣行业演进趋势、区域产业需求与高等教育发展规律，对规划文本进行了深入剖析与论证。专家们围绕发展定位、学科专业布局、人才培养模式、师资队伍建设、产教深度融合及保障机制等关键维度，逐一提出具有前瞻性、针对性与建设性的修改意见。专家指出，规划应进一步凸显学院特色优势，强化与行业产业的同频共振与深度对接；进一步优化目标指标体系，增强实施路径的科学性与可操作性；确保规划既立足当前现实基础，又着眼长远战略布局，既遵循教育发展规律，又紧密贴合时代发展脉搏。
随后，与会人员聚焦规划中的重点领域与难点环节，与专家们进行了深入的交流研讨，现场气氛热烈、务实高效，进一步碰撞了思想、凝聚了共识。
此次专家咨询会的顺利召开，是对学院“十五五”规划编制工作的一次精准把脉与有力赋能。学院将认真梳理、充分吸纳专家的真知灼见，进一步打磨完善规划文本，切实以高水平规划引领高质量发展，为学院“十五五”期间乘势而上、开启发展新篇章奠定坚实基础。
