# 武汉大学人工智能学院2026年普通招考博士研究生（第二批次）综合考核工作通知
根据《武汉大学人工智能学院2026年普通招考博士研究生招生工作实施细则》，现将我院2026年普通招考博士研究生（第二批次）综合考核相关事项通知如下：
一、组织管理
我院研究生招生工作领导小组全面负责我院2026年普通招考博士研究生（第二批次）综合考核工作。
二、招生计划
学院2026年第二批次计划招收全日制博士研究生21人，其中学术学位博士研究生20人，专业学位博士研究生1人。
**学院2026年博士招生计划实际录取人数根据最终下达招生计划数及生源情况可能调整，以实际拟录取结果为准。**
三、外语综合水平考核
我院根据外语水平免试条件进行审核，免试考生审核结果已在博士招生管理系统中公布，非免试考生须参加2026年4月18日武汉大学博士外语水平考试，成绩达到50分（此项工作已完成）。
四、候选人选拔
我院考核专家组根据考生提交的材料，对考生的学习能力与学科基础、研发能力与实践创新成果、培养潜力进行审核评议，做出综合评价结论。依据综合评价成绩，按不超过招生计划数1∶3的比例确定可以参加综合考核的考生，即候选人，报研究生院备案后进行公示（评估成绩及考核候选人信息见附件1）。
确定参加综合考核的候选人请于2026年5月15日中午12:00前实名申请加入“WHUAI2026年博士综合考核群（第二批次）”**（QQ群号：253462343）** 。入群验证申请格式：“姓名－报考专业”，后续通知会在群内发布，请及时关注群消息。
确认放弃参加综合考核的候选人请发邮件至rqhuang@whu.edu.cn，邮件主题为“姓名－报考专业－放弃综合考核”。
五、综合考核
（一）综合考核方式
采取线下方式进行。
（二）综合考核时间安排
2026年5月17日 14:00—15:00 报到、资格审查
2026年5月17日 16:00—18:00 综合笔试
2026年5月18日 09:00—16:00 综合面试，具体安排QQ群内通知，请保持通信畅通。
（三）现场报到及资格审核
参加综合考核的候选人在报到时须提供以下材料：
1. 往届生（已取得硕士学位人员）须交验本人最后学位、学历证书和身份证等原件；应届硕士生须交验硕士研究生学生证、学信网学籍认证报告和身份证等原件；
2. 在境外教育科研机构获得硕士学位、学历的考生，须出具教育部留学服务中心的认证书原件或者学校出具的预计毕业时间的证明原件。
3. 提交以下材料复印件：
（1）按顺序装订（左上角）：身份证（正反面复印在同一页面），最后学历证书、最后学位证书（应届硕士生提交学生证及学信网学籍认证报告；获得境外学位、学历的考生，提交教育部留学服务中心的认证书复印件或学校出具的预计毕业时间证明复印件）；
（2）单独提交：《2026年武汉大学博士研究生考生信息卡》，须本人仔细核对，并粘贴照片及签名。
（3）单独提交：《考生诚信综合考核承诺书》（附件2），需本人亲笔签名。
4. **考生须于报到时携带U盘，将个人汇报PPT拷贝至工作人员处，报到结束后不再接受拷贝或更换** 。
5. 综合考核收费180元/人，收费办法及要求由培养单位按学校有关规定执行。
学院将核对考生的报考信息，按报考条件严格审查考生报考资格，资格审核未通过的考生不得参加综合考核。不符合报考条件或提供虚假信息者，取消其综合考核资格。考生未按要求到现场报到或参加综合考核者，视为自动放弃综合考核资格。
（四）综合考核内容
综合考核内容主要包括综合笔试和综合面试。跨一级学科报考者不加试。
1. 综合笔试：学院组织的综合笔试主要考查学科基础水平，考试时间为2小时，满分为100分，笔试采用闭卷方式。
2. 综合面试（含英语听说能力测试）：学院结合考生报名情况分组开展面试。
（1）外语水平（不超过3分钟）。在面试中进行英语听力口语测试。主要考查候选人是否具备本学科博士研究生应具备的外语应用能力。候选人须准备英语自我介绍，并回答专家提问。
（2）个人汇报（不超过7分钟）。考生向考核专家组汇报本人工作、科研及技术研发经历、相关成果、对拟从事研究的技术研发领域及研究方向的认识、研究思路和展望。报告采用PPT形式。
（3）考核专家组对候选人基础知识、对本学科前沿知识了解情况、综合运用所学知识分析问题、解决问题的能力及科技创新的敏锐度、思想政治素质和品德、对攻读博士的认识等综合素质进行全面考查。综合面试时间约20分钟。
（五）综合考核总评成绩计算
综合考核总评成绩 = 笔试成绩×30% + 面试成绩×70%
综合考核总评成绩、综合笔试成绩、综合面试成绩均为百分制。
（六）录取原则
坚持“择优录取、保证质量、宁缺毋滥”原则，学院招生工作领导小组根据招生计划、考生综合考核总评成绩按照专业类别确定拟录取名单并公示。
**综合考核总评成绩不合格者（低于60分）不予录取。**
思想政治素质和品德考核内容包括考生的政治态度、思想表现、道德品质、遵纪守法、诚实守信、心理健康等方面。学院将严格遵循实事求是的原则认真做好考核工作。**思想政治素质和品德考核不合格不予录取。**

人工智能学院黄老师：027-68765299（工作时间开通）
