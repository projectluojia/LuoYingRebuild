# 武大本科招生六大亮点，每一款都适合你！

栏目：热点武大

2024年是武汉大学的 “人才培养年” ，学校聚焦拔尖创新人才培养，加快推进综合改革，AI赋能教学，数智课程全覆盖，以雷军班、数学中法班为代表的 交叉人才培养、衔接培养试验班 蓬勃发展。

一、招生规模适度扩大，优质热门专业计划显著增加

2024年，武汉大学130个本科专业，按照41个专业（类）面向全国招生7215人，学校招生总人数有所增加，强基计划招生专业从8个增加到9个。学校加大招生-培养-就业联动力度，动态调整各专业招生计划，加快推进衔接式培养模式改革，加大跨学科交叉人才培养力度，促进人才培养结构与国家需求、社会需要相适应。2024年， 弘毅学堂、法学、计算机、遥感、电气等优质热门专业招生计划数显著增加 ，在各省都安排了招生计划，并且相比往年还有所增加。在高考招生录取环节，充分尊重考生意愿， 近三年考生专业满足率均超过98% ，预计2024年还将更高，更好满足群众“上好学”的需求。

二、实施拔尖行动计划，建设三级特区培养拔尖人才

2023年11月，武汉大学发布“拔尖创新人才培养行动计划”，发挥武汉大学基础研究扎实、学科交叉融合和创新氛围浓厚的综合性研究型大学优势，强化有组织拔尖创新人才培养，着力培养担当民族复兴大任的新时期本科拔尖人才。

学校以“规模精、质量高、国际化”为指导，积极打造拔尖创新人才培养“三级特区”，探索本研贯通、跨学科、主辅修等人才培养方式，具备更加个性化的培养方案、更高水平的师资团队、更交叉的课程体系、更高的升学深造比例。2024年， 武汉大学共建设了58个特色试验班 ，以拔尖计划2.0基地、“强基计划”招生专业和国家级特色化示范性软件学院为依托， 建设了16个国家级特区 ；结合国家发展需要和学校专业特色 ，建设了6个校级示范区 ； 此外还有36个院级试验区 。 其中31个衔接试验班采用本研贯通培养模式 ，打通培养通道，助力学生成长成才； 29个试验班直接面向高考直接招生，23个试验班进校后二次选拔 。

三、构建数智教育体系，人人可以学习大数据和AI

武汉大学顺应数字时代潮流，响应国家战略规划与需求，于2023年11月17日率先发布数智教育白皮书，提出建设具有武汉大学特色的一体化数智人才培养体系，将数智人才培养覆盖到所有学科专业及各个学历层次，在推动数据科学知识的交叉学习基础上，更加注重数字技能的融通实训，着力培养面向未来的复合型数智人才。

数智教育是以大数据和人工智能技术为主要载体，培养具备数字思维与数字素养的交叉人才。 武汉大学在数智教育领域有得天独厚的条件，遥感专业连续七年世界第一，信息管理、管理、法学、计算机等专业实力雄厚，有丰富的数智教育经验 。目前，学校已建设1门“数智+”基础通识课程——“人工智能导引”和16门特色数智教育系列课程，形成了系统化的数智教育体系。

四、弘毅学堂扩容升级，雷军班招生，数学中法班二选

2024年，武汉大学拔尖人才三级特区中有三个试验班改革力度很大——弘毅学堂试验班更加突出交叉特色，雷军班更加突出产教协同育人特色，数学中法班更加突出国际化培养特色。

一是弘毅学堂扩容升级 。弘毅学堂是武汉大学的荣誉学院、拔尖人才培养高地。 2024年，弘毅学堂招生人数大幅增加，从245人增加到400人，增幅超过60% ；学堂对原有试验班升级， 设立了9个试验班(PPE试验班、PLE试验班、数字经济试验班、数理经济与数理金融试验班、全球治理试验班、大健康试验班、智能制造与集成电路试验班，未来科技试验班、人文科学试验班)，每个试验班都是2-3个专业方向交叉人才培养。弘毅学堂按3个专业招生 ，进校后， 学生可以按照个人兴趣爱好，在9个试验班中任意选择 。进入到未来科技试验班、人文科学试验班的学生，对基础学科感兴趣的，一年后根据自己的兴趣爱好，进入数学、物理、化学、生物、文学、历史、哲学等学院继续拔尖计划培养；其中，获得校长特别奖和部分高分考生，一年后可全校专业任选（医学长学制除外）。

二是雷军班首次招生 。雷军班是武汉大学探索多元化人才培养的重要举措，由雷军校友亲自领衔，旨在培养未来计算机学科的创新创业领军人才。2024年， 雷军班计划招生30人，其中一半对高考招生，一半面向全校新生二次选拔 。雷军班强化校企共建，产教深度融合，产业导师、学业导师和工程导师“三师共育”，实行更加灵活的课程学分认定和评价制度，包括部分课程免修免考、弹性学制和科研轮转机制等。 学生可以自主定制大学学习，达到学业标准的学生可100%保研100%直博 。

三是数学中法班恢复招生 。武汉大学数学中法班诞生于上世纪80年代，在中法两国政府支持下，武汉大学与法国知名高水平大学在数学拔尖人才培养方面开展的卓有成效的合作，培养出大批具有国际影响力的顶尖优秀数学家。该项目办学模式成为中国教育改革的典范，育人成果备受国际数学界赞誉。 2024年武汉大学数学中法班恢复招生，每届招生人数不超过30人，面向全校新生二次选拔 。数学中法班高水平师资团队由张平文等院士领衔，中方和法方名师组成教学委员会全程指导教学，培养方案参照法国数学本科专业大纲设置，数学专业核心课程由法国知名学者授课，强化法语课程， 可授法语辅修双学位。全部学生可申请出国深造，学校每年选派7名优秀中法班毕业生赴法国深造攻读博士 。高水平师资推荐中法班学生赴法国高校继续深造。

五、更加尊重学生意愿，把选择权交给学生

武汉大学充分尊重学生意愿，引导学生找到与自身能力和兴趣相匹配的专业方向。2024年，学校明确了“学院内转专业无限制，同一学部内尽量满足，不同学部间适度满足”的原则，对原有转专业管理办法进行修订，发布了《武汉大学普通本科生转学院管理办法（试行）》和《武汉大学普通本科生大类分流和专业调整管理办法（试行）》，自2024级学生起开始实施。此举旨在激发学生学习主动性和创造性，促进学生个性发展。

学校明确， 一是学院内专业100%满足 。大类分流和专业调整应充分尊重学生兴趣和意愿，原则上不得设置限制条件。时间延长到大三下学期结束前，院内专业调整工作原则上每学年一次，学生从大一到大三均可申请。 二是转出学院100%不受限 。秉持学生为本理念，学院不得限制学生转出学院的申请，学校不控制申请转出人数，确保学生在学部内转学院时能够尽可能满足，仅在跨学部转学院时采取保护性限制，避免因学科差异过大而影响学生学业。学生可以根据自身兴趣和专长申请转学院，最大限度地满足学生个性发展的需求。

六、校友助力人才培养，“领航计划”引领学生成长

多年来，武汉大学培养了一大批优秀的毕业生和众多杰出校友。武汉大学校友是最爱母校的校友，校友们为学校发展慷慨解囊。2023年11月，学校130周年校庆时，武大校友捐赠超过20亿元。得益于校友的支持， 学校在本科生中设立了50余项奖学金，受奖面占学生总数的30%以上，本科生最高可以一次获得10万元奖学金、15万元境外学习资助 。

2024年， 学校将实施知名校友企业家导师“领航”计划 。对于2024年录取到武汉大学的各省优秀新生，学校将选派武汉大学知名校友企业家担任导师，“一对一”指导。校友企业家导师引导学生更好适应大学学习生活，拓宽眼界，做好职业规划，确保实习和就业，提供海外研学机会，并每月安排一场名师论坛，邀请各行各业大咖来校作报告，助力学生更好成长。
